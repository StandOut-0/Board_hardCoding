# Board_hardCoding

#### 게시판<br>
실 프로젝트 진행 전 간단히 CRUD를 구현해보며 전체 틀을 이해하기 위함을 목적으로 한다.<br>
일정 및 기획은 NOTION에 간단히 기록하며 진행하고<br>
과정은 기존 운영하던 블로그에 기록한다.<br>
